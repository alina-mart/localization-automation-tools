# Localization Automation Tools

A showcase of automation tools and validation systems built for localization and content production workflows in the gamedev industry.

## Technologies
- Google Apps Script (JavaScript)
- Google Sheets advanced formulas
- Regex

## Examples included
- HTML export parsing automation
- Text change detection
- Glossary compliance validation
- Source/target consistency checks
- Reporting automation

## Purpose
This repository contains demo/sanitized examples inspired by production workflow automation and QA tooling.
